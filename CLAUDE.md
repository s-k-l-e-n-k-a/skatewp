# Skate — Claude Instructions

## Workflow

- **No hacer commits ni push automáticamente.** Solo cuando el usuario lo pida explícitamente.
- Flujo: editar → probar en localhost → commit solo cuando el usuario lo pida → push solo cuando el usuario lo pida.

## Language

- **All backend UI text in English** — labels, headings, buttons, placeholders, notices.
- **All code comments and documentation in English** — inline comments, docblocks, README content.
- Exception: user-facing frontend text (if any) follows the site language.

## Stack

- **PHP** — lógica de backend, shortcodes, hooks de WordPress
- **Vanilla JS** — sin jQuery, sin frameworks; JS nativo para toda interactividad
- **SCSS** → compilado a CSS con `npm run build:css` (dart-sass + PostCSS/autoprefixer)
- **WordPress API** — usar siempre las APIs nativas de WP cuando existan:
  - `wp_options` para persistencia de datos
  - `wp_kses()` para sanitización de SVG/HTML
  - Variables CSS de theme.json (`var(--wp--preset--color--*)`, `var(--wp--preset--font-size--*)`) en lugar de valores hardcodeados
  - Hooks, nonces, capabilities para seguridad
  - `get_permalink()`, `home_url()`, etc. para URLs
- **Sin Composer, sin bundler de JS** — el tema no usa dependencias externas de PHP ni empaquetadores JS

## Admin Menu Order

The Skate admin pages must always appear in this order — both in the WP left sidebar and in the Skate topbar:

1. Settings
2. Identity
3. Design
4. Navigation
5. Sitemap

Maintain this order whenever adding, removing, or renaming admin pages. The sidebar order is enforced via a late `admin_menu` hook (priority 999) in `inc/backend/general.php`. The topbar order is the `$nav` array in `skate_admin_topbar()`.

## CSS Class Prefix for User-Applied Classes

Classes intended to be added manually by users in the WordPress editor (block Advanced settings → Additional CSS class) use the prefix **`pace-`** (not `skate-`). This makes them version-agnostic — they remain valid if the theme is renamed or versioned to Pace5, etc.

Examples: `pace-submenu-featured-title`, `pace-no-shadow`

- `skate-` prefix is reserved for internal theme classes (generated by PHP/SCSS, not set by users).
- `pace-` prefix is for user-facing utility classes documented in the UI.
